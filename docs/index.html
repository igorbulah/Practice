Принцип работы

Воздушно-пузырьковая панель представляет собой полую конструкцию из акрилового стекла, заполненную жидкостью. Сложная система, состоящая из нескольких компонентов, обеспечивает непрерывную подачу воздуха, создавая поток пузырьков. Воздушные пузырьки за счет сопротивления воды образуют удивительные потоки и вихри.

Монтаж

Монтаж панели следует производить в чистых хлопчатобумажных перчатках после окончания основных отделочных работ. Перчатки предохранят глянцевую поверхность акрила от случайных отпечатков пальцев.

Уход

Идеальный материал для ухода - тряпка из микрофибры. Порядок мытья: сначала легкая влажная протирка, затем легкая финишная – насухо для удаления разводов. Давление на материал должно быть минимальным.

Мытье

Для мытья панели следует использовать только теплую чистую воду. Жирные и стойкие загрязнения удаляются с помощью мыльной воды. Не рекомендуется использовать спиртосодержащие моющие средства!

Типы водно-пузырьковых панелей

В зависимости от вида образуемого эффекта от движения пузырьков панели подразделяются на несколько видов: панели с одиночными крупными пузырьками; с множественными мелкими пузырьками; с комбинированным эффектом одновременное сочетание пузырьков разного размера). По типу панели делятся на: сотовые пузырьковые секции, монопанели (вихревая), комбинированные конструкции

По типу панели делятся на: сотовые пузырьковые секции, монопанели (вихревая), комбинированные конструкции

Сотовые пузырьковые секции

Делают из акрилового стекла, имеющего сотовую структуру, то есть состоят из длинных замкнутых сот. Толщина панели наиболее распространенного акрилового листа марки «Plexiglass» составляет 16 мм, а ширина одной соты-ячейки – 64 мм.

Монопанель (вихревая)

Представляет собой один большой сосуд с водой (без сот). Пузырьки воздуха, выходящие из форсунок свободно перемещаются, создавая вихревые потоки. При пересечении потоков друг с другом образуются захватывающие абстрактные динамические изображения.

Комбинированные конструкции

Системы, сочетающие пузырьковый эффект с другими видами водных систем. Например, можно совместить в одну композицию пузырьковые трубы и искусственный водопад по струнам.

Вопрос-ответ

Можем ли мы использовать большие пузырьковые панели в фойе ?

Да, можете. Но в этом случае необходимо чаще производить плановую очистку системы.

Возможно ли индивидуальное изготовление водопада или панели с нашими фирменными знаками?

Да, конечно! Смотрите примеры таких работ. Срок изготовления изделия - от 7 рабочих дней.

Можно ли использовать обычную воду?

По условиям эксплуатации, можно использовать только дистиллированную воду. Обычная вода со временем будет давать разводы и отложения на стекле.
